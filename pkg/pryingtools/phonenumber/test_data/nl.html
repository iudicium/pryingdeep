Dutch Phone Number Validation

Valid Dutch Phone Numbers:

  • +31636363634
  • 0031 612345678
  • 0 70 1234567
  • 06-36363634
  • 06 36363634
  • 06-3636-3634

Invalid Dutch Phone Numbers:

  • +3112345678 (Invalid country code)
  • 06123456789 (Invalid mobile number length)
  • 0123456789 (Invalid area code)
  • 0031-10-1234-5678 (Invalid separators)
